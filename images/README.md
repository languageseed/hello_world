# Images Folder

Place your blog post images here!

## Supported Formats

- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- WebP (.webp)
- SVG (.svg)

## Using Images in Posts

In your markdown files, reference images like this:

```markdown
![Alt text description](../images/your-image.jpg)
```

## Tips

1. **Optimize your images** before adding them (compress to reduce file size)
2. **Use descriptive filenames** (e.g., `sunset-beach-2025.jpg` instead of `IMG_1234.jpg`)
3. **Add alt text** for accessibility and SEO
4. **Keep organized** - consider subdirectories for different topics or dates

## Example

If you have an image at `images/hello-world.png`, use it in your post like this:

```markdown
![Hello World Banner](../images/hello-world.png)
```

The image will automatically be:
- Responsive (scales to fit the screen)
- Beautifully styled with rounded corners
- Shadow effects for depth
- Properly spaced

## Image Captions

While not built-in to standard Markdown, you can add captions using italic text:

```markdown
![My Image](../images/my-image.jpg)

*This is a caption for my image*
```
