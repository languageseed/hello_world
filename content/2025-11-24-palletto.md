## How It Works

Palletto uses smart color extraction:

1. **Downscales** images to ~150px for efficient processing
2. **Extracts** unique colors with quantization (reduces near-duplicates)
3. **Clusters** using k-means++ algorithm for optimal color grouping
4. **Weights** colors by frequency for accurate representation
5. **Transforms** into 11 variations by manipulating OKLCH components (Lightness, Chroma, Hue)

Export your palettes as CSS custom properties ready to drop into any project:

```css
:root {
  /* Extracted Colors */
  --primary-1: oklch(0.5 0.2 270);
  --primary-2: oklch(0.6 0.15 330);
  
  /* Soft Pastels */
  --soft-1: oklch(0.85 0.08 270);
  
  /* Complementary */
  --inverted-1: oklch(0.5 0.2 90);
  
  /* ... all 11 palettes */
}
```
