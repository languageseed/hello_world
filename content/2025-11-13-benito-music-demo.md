## Try It Yourself!

Want to add your own audio to the blog?

### 1. Add Your Audio File

```bash
cp your-audio.wav audio/
```

### 2. Convert to HQ MP3

```bash
python scripts/convert_audio.py --force
```

### 3. Create a Post

```markdown

```

### 4. Publish

```bash

git add .
git commit -m "Add audio post"
git push origin main
```
