üöÄ Quick Start Guide

  1. Create a new markdown file (e.g., my-post.md)
  2. Add frontmatter with title, author, and date
  3. Write your content using Markdown syntax
  4. Run: python generate_post.py my-post.md
  5. Your HTML post will be generated in the posts/ folder
  6. Add your post to this index page manually or use the update script
